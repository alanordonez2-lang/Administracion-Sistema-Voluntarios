# Administracion-Sistema-Voluntarios

# Descripción

Este proyecto contiene la resolución de dos ejercicios desarrollados en consola utilizando los principios de la Programación Orientada a Objetos (POO) en C#.

# Los ejercicios fueron diseñados para aplicar conceptos fundamentales como:

Herencia.
Polimorfismo.
Encapsulamiento.
Reutilización de código.
Separación de responsabilidades.
Diseño orientado a objetos.

# Descripción de Sistema

Desarrollar un sistema de consola que permita administrar las personas que forman parte de una organización solidaria.

El sistema debe gestionar distintos tipos de integrantes, permitiendo registrar su información, mostrar sus datos y aplicar comportamientos específicos según el rol que desempeñan dentro de la organización.
